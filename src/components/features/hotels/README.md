# Hotel Management Components

Components for hotel listing, management, and onboarding.

## Components
- `hotel-card.tsx` - Display hotel information
- `add-hotel-form.tsx` - Create new hotel
- `add-hotel-info-form.tsx` - Hotel information step
- `add-hotel-facilities-form.tsx` - Hotel facilities step
- `add-hotel-documents-form.tsx` - Hotel documents step
- `add-hotel-rooms-form.tsx` - Hotel rooms step
- `similar-properties.tsx` - Similar hotel suggestions
- `image-grid.tsx` - Hotel image gallery

## Folders
- `hotel-onboarding/` - Multi-step hotel onboarding flow

## Usage
```typescript
import { HotelCard, AddHotelForm } from '@/components/features/hotels'
```
