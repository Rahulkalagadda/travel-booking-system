# Search Components

Hotel and travel search functionality.

## Components
- `search-form.tsx` - Main search form
- `search-filters.tsx` - Filter options
- `search-suggestions.tsx` - Search autocomplete
- `SearchBar.tsx` - Quick search bar
- `FilterSidebar.tsx` - Sidebar filters

## Usage
```typescript
import { SearchForm, SearchFilters } from '@/components/features/search'
```
