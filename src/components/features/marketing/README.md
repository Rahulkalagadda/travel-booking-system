# Marketing & Landing Page Components

Components for the marketing website and landing pages.

## Components
- `hero-3d-scene.tsx` - 3D hero animation
- `hero-scene-wrapper.tsx` - Hero scene container
- `marketing-sections.tsx` - Marketing content sections
- `plan-your-trip.tsx` - Trip planning section
- `verticals-showcase.tsx` - Service verticals display
- `meet-the-host.tsx` - Host introduction section

## Usage
```typescript
import { HeroSceneWrapper, MarketingSections } from '@/components/features/marketing'
```
